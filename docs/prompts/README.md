# Prompt Library

## Purpose
Standardized prompts for architecture, backend, frontend, database, testing, security, review, refactoring, and documentation.

## Structure
```
docs/prompts/
  Architecture/
  Backend/
  Frontend/
  Database/
  Testing/
  Security/
  Review/
  Refactoring/
  Documentation/
```

## Rules
- All team and AI agents use approved prompts from this library.
- Prompt changes require review and version bump.
- Prompts must reference applicable `docs/standards/` documents.

## Compliance
Every AI-assisted sprint must cite prompt IDs used in the ADR or sprint report.
