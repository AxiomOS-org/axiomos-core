# ADT Architecture

## Input
- Requirement spec (human or AI planner)
- Standards profile
- Target module metadata

## Processing
1. Planner builds blueprint.
2. Architecture validator checks freeze rules.
3. Generator pipeline resolves templates.
4. Quality checker validates output readiness.

## Output
- Generated artifact set
- Validation and quality reports
- Documentation bundle

## Extensibility
- Plugin and marketplace extension points.
- Transformer chain hooks for evolution.
