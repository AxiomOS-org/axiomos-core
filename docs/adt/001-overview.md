# 001 ADT Overview

## Objective
Define AxiomOS Developer Toolkit (ADT) as an enterprise engineering pipeline, not only a code generator.

## Scope
- Documentation-first design for ADT.
- No implementation in this phase.

## ADT Vision
Business Requirement -> AI Planner -> Architecture Validator -> Generator -> Static Analysis -> Tests -> Documentation -> Review -> Ready Module

## Outcomes
- Standardized module delivery.
- Reduced boilerplate errors.
- Higher governance compliance.
- Faster, safer module development.
