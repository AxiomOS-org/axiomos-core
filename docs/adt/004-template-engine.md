# 004 Template Engine

## Responsibilities
- Resolve template sets per module type.
- Merge user inputs with defaults and standards.
- Render file trees consistently.

## Requirements
- Versioned templates.
- Strict schema validation before render.
- Support conditional sections by capability flags.

## Extensibility
- Third-party template packs via plugin SDK.
- Compatibility matrix tied to `minimumCoreVersion`.
