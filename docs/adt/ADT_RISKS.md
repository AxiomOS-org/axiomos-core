# ADT Risks

## Key Risks
1. Template drift from evolving standards.
2. Over-generation causing low-value boilerplate.
3. False confidence if quality checker is incomplete.
4. Compatibility issues across toolkit versions.
5. AI planning hallucinations on ambiguous requirements.

## Mitigations
- Versioned template governance.
- Mandatory architecture validation stage.
- Incremental rollout by generator type.
- Golden-module regression suite for ADT.
