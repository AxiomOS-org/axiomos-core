# 015 Security Rules

## Baseline Controls
- Authentication and authorization are mandatory for protected endpoints.
- Principle of least privilege for users, services, and modules.

## Data Protection
- Never log secrets, tokens, or credentials.
- Hash passwords using current secure defaults.
- Encrypt sensitive data at rest where required by policy.

## Operational
- Security tests are required in quality gate.
- Vulnerability findings must be triaged with SLA by severity.
